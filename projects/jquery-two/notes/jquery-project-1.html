Creating Elements Project

Setup

In the jquery-two project directory, open the index.html file, and RIGHT ABOVE the closing-body-tag, add a script tag with a src of "script.js", like this:

    <!-- OTHER MARKUP -->

  </body>
</html>

In the same directory, create a script.js file with the following contents:

$(function(){
  var avengers = ["Iron Man", "Thor", "Hulk", "Ant Man", "Wasp"];

  // Code goes here!

});

All the exercise code moving forward will go in this script.js file.

Exercise 1 : Avengers Assemble!

  1. Write and call a function that expects an array as an argument. Within this function, append a <ul> to the DOM with the ID avengers.
  2. Next, still within this function, write a loop that creates an <li> and appends it to the ul for each item in the array. For each item, add a class corresponding to the item's content. For example, the first item in the array should have an element that looks like <li class="iron-man">Iron Man</li>.
Hint

.split(), .join(), .toLowerCase() and .replace() are some useful functions you might investigate.

Exercise 2: Avengers Reassemble!

  1. Write and call a new function that removes the <li>s for Ant Man and Wasp, and adds an <li> for Captain America.

Extensions!

  1. Sort the array alphabetically before adding the elements to the DOM
  2. Sort the elements alphabetically after they're in the DOM
  3. Create a button that moves the first element to the bottom of the list when clicked
  4. Add Set Timeout so that items appear on the page one at a time.
