In the script.js file that is linked to the index.html page, target some DOM elements and apply some of these jQuery convenience methods to animate your elements: (hint: jQuery API)

  1. .fadeIn() - fade in a set of matched elements over time
  2. .fadeOut() - fade out as set of matched elements over time
  3. .fadeTo() - fade a set of matched elements to a specific opacity over time
  4. .slideDown() - animate an element sliding down over time
  5. .slideUp() - animate an element sliding up over time
  6. .slideToggle() - toggle between an element sliding up or down over time depending on whether it's visible or not

Reference: Event handling

  1. 'blur' - this event is emitted when an element loses focus, which happens when the user has focus on an element and Tabs to another element or clicks elsewhere
  2. 'change' - this event is emitted when the value of an input field is changed, such as a text area, text input field, radio button, etc.
  3. 'click' - this event is emitted when an element is clicked on
  4. 'dblclick' - this event is emitted when an element is double-clicked on
  5. 'error' - this event is emitted when an element is not loaded correctly, such as elements
  6. 'focus' - this event is emitted when an element gains focus, meaning that the user has reached the element via the Tab key or has clicked on it
  7. 'hover' - this event is emitted when the mouse pointer enters or leaves the element
  8. 'keydown' - this event is emitted when the user first presses any key down
  9. 'keyup' - this event is emitted when the user releases a key
  10. 'mousedown' - this event is emitted when the user presses a mouse button over the targeted element
  11. 'mouseup' - this event is emitted when the user releases a mouse button over the target element
  12. 'mouseenter' - this event is emitted when the user moves the mouse pointer into the targeted element
  13. 'mouseleave' - this event is emitted when the user moves the mouse pointer out of the targeted element
  14. 'mousemove' - this event is emitted when the user moves the mouse around within the targeted element
  15. 'resize' - this event is emitted when the targeted element is resized
  16. 'scroll' - this event is emitted when the target element is scrolled
  17. 'select' - this event is emitted when the target element is selected, usually only used for various types of inputs
  18. 'submit' - this event is emitted when the target element is submitted, and can only be attached to form inputs
