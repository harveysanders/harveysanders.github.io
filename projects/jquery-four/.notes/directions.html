Exercise 2 : Consuming Data Services

Loading data for consumption is a typical pattern in software applications. Often, the user makes a request to a service, either to a web server exposing a REST API that returns data, or we may load local JSON for various reasons. Making use of jQuery, we can display the results of these requests in a dynamic manner.

Setup:

  • In your website's workspace, in the jquery-four project, open up the index.html page - we'll code in the <script> tag of this page.
  • You may already have the index.html page open in your browser, but if not, click here to open the index.html page in your browser...

Consume and Present Data:

  • Follow the numbered TODO's (TODO 1, TODO 2, etc...) within the <script> tag. For each service we invoke on a click-event, there's an associated initialize function - within these initialize functions, you can pretty-up the data returned from the service to make it more presentable.
