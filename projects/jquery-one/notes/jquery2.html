Getting jQuery

There are two ways to get jQuery:

  1. Go to the jQuery site, download it, and put the relative link in your HTML
  2. Put the absolute link to the asset on the jQuery site

Let's first download jQuery and then talk about the two different methods. You'll need to set up an empty project folder for your HTML file and scripts.

Link it

Normally you don't want to link to jQuery; it creates a dependency to a 3rd party website, creating these downsides:

  • You will need the internet just to test locally
  • The 3rd party website might go down, and thus break your app

However, it's conveniently quick for getting small projects / exercises up and running. Here are the steps:

  1. Go to https://code.jquery.com/
  2. Find the "jQuery core 2.x" section
  3. Right-click and copy the link (usually you will want the uncompressed version)
  4. Paste the link into a script tag on your page.

Your script tag will look something like the following:

1
 
<script src="https://code.jquery.com/jquery-2.1.1.js" type="text/javascript"></script>

Download it

To get around the downsides in the previous section, you can download jQuery so you rely only on your own server (or, during development, your own computer).

Here are the steps:

  1. Go to http://jquery.com/
  2. Navigate to the download page
  3. Download the uncompressed, development version of jQuery 2.x
  4. Put the file inside your project folder

Next, let's create an index.html file, and set up a script tag inside of it referencing our new file (your version number may be different):
